# Personal CV Portfolio

A local portfolio website that works like a personal LinkedIn-style profile.

## What it has

- Hero section, profile summary, and contact links
- Education, work experience, skills, projects, and future projects
- Dark black and blue theme
- Editable content from one JSON file
- Built-in editor at `/edit`

## Run

```powershell
cd C:\Users\qamar\cv-portfolio
C:\Users\qamar\AppData\Local\Programs\Python\Python314\python.exe app.py
```

Open:

- `http://127.0.0.1:8010`
- `http://127.0.0.1:8010/edit`

## Edit your content

You have two ways to update the site:

1. Open `/edit`, change the JSON, and save.
2. Edit `content.json` directly.

The layout code stays in `app.py` and `static/styles.css`.
